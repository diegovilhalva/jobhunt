# Job Hunt 🧑‍💼💼

**Job Hunt** is a full-stack job portal web application where users can explore job opportunities, apply for them, and recruiters can post and manage job listings. It was built using the **MERN Stack** and deployed on **Vercel**.

## 🌐 Live Demo

👉 [https://jobhunt-jh38.vercel.app/](https://jobhunt-jh38.vercel.app/)

## 📂 Repository

Source code: [https://github.com/diegovilhalva/jobhunt](https://github.com/diegovilhalva/jobhunt)

---

## ✨ Features

### 👤 Job Seekers

* Browse and search for jobs by keyword, category, or location.
* View job details.
* Apply for jobs.
* Track the status of your applications (Applied, Under Review, Rejected, etc).

### 🧑‍💼 Recruiters

* Protected dashboard with role-based access control.
* Create, edit, and delete job postings.
* View all applicants for each job.
* Update the application status for each candidate.

### 🔐 Authentication

* JWT-based sign up and login.
* Role-based route protection (only recruiters can access certain pages).

---

## 🛠️ Tech Stack

### Frontend

* React + Redux Toolkit
* Vite
* Tailwind CSS + shadcn/ui
* Lucide-react icons
* Axios
* React Router

### Backend

* Node.js + Express.js
* MongoDB + Mongoose
* JWT Authentication
* CORS + dotenv
* RESTful API architecture

## 🚀 Getting Started (Development)

### Prerequisites

* Node.js
* MongoDB (local or MongoDB Atlas)

### Clone the Repository

```bash
git clone https://github.com/diegovilhalva/jobhunt
cd jobhunt
```

### Backend Setup

```bash
cd backend
npm install
npm run dev
```

Create a `.env` file with the following:

```env
PORT=4000
MONGO_URI=your_mongodb_connection
JWT_SECRET=your_jwt_secret
```

### Frontend Setup

```bash
cd frontend
npm install
npm run dev
```

## 🚀 Getting Started (Development)

### Prerequisites

* Node.js
* MongoDB (local or MongoDB Atlas)

### Clone the Repository

```bash
git clone https://github.com/diegovilhalva/jobhunt
cd jobhunt
```

### Backend Setup

```bash
cd backend
npm install
npm run dev
```

Create a `.env` file with the following:

```env
PORT=4000
MONGO_URI=your_mongodb_connection
JWT_SECRET=your_jwt_secret
CLOUDINARY_NAME=your_cloudinary_cloud_name
CLOUDINARY_API_KEY=your_cloudinary_api_key
CLOUDINARY_API_SECRET=your_cloudinary_api_secret
```

### Frontend Setup

```bash
cd frontend
npm install
npm run dev
```

Create a `.env` file with the following:
```env
    VITE_API_ENDPOINT=http://localhost:4000/api/v1
```
